# Configuración de Supabase

1. Crea un archivo `lib/supabase_config.dart` con el siguiente contenido (no lo subas a repositorios públicos):
	```dart
	const String supabaseUrl = 'TU_SUPABASE_URL';
	const String supabaseAnonKey = 'TU_SUPABASE_ANON_KEY';
	```
2. Obtén estos valores desde tu panel de Supabase (Project Settings > API).
3. El archivo ya está referenciado en `main.dart` para inicializar el servicio.

Recuerda mantener este archivo fuera del control de versiones si tu proyecto es público.
